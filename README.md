# Планировщик задач (Локальный TODO-сервер)

## Описание проекта

Данный проект создан для упрощения отслеживания поставленных задач и запланированных событий.
В нем присутствуют функции добавления, удаления, редактирования, настройки повторений задач и событий, а так же возможность
отметить их завершенными.

## Инструкция по запуску кода (локльно)

1. Установить Golang v1.22+
2. Клонировать репозиторий с GitHub в корневую директорию свой ПК
3. Открыть командную строку bash и ввести следующие команды:
```bash
    # Перейти в директорию проекта:
    cd sprint_13-14_final_project

    # Установить зависимости:
    go mod tidy

    # Запустить локальный сервер:
    go run main.go
```
4. В браузере открыть http://localhost:7540/

## Инструкция по запуску тестов
1. Убедиться, что сервер запустился без ошибок
2. Для запуска тестов использовать команду:
```bash
    # Запуск всех тестов
    go test ./tests
    # Тест добавления задач
    go test -run ^TestAddTask$ ./tests
    # Тест удаления задач
    go test -run ^TestDelTask$ ./tests
    # Тест получения задач
    go test -run ^TestTasks$ ./tests
    # Тест редактирования задач
    go test -run ^TestTask$ ./tests
    # Тест расчета следующих дат
    go test -run ^TestNextDate$ ./tests
    # Тест выполнения задач
    go test -run ^TestDone$ ./tests
```
